# Práctica 3 - Programación 2

`Red Neuronal Convolucional para MNIST`
---


`definición:` Una Red Neuronal Convolucional es una arquitectura de aprendizaje profundo diseñada para procesar datos con estructura de grilla, como imágenes.

Ventajas de las `CNNs`
- Detectan bordes, texturas, y formas automáticamente.
- Requieren menos procesamiento que técnicas tradicionales.
- Escalables a grandes volúmenes de imágenes.

`objetivo de la práctica:`

Comprender y aplicar una red neuronal convolucional para el reconocimiento de imágenes utilizando el conjunto de datos MNIST.

Temas abordados:
- Arquitectura de redes convolucionales.
- Clasificación de imágenes.
- Evaluación de rendimiento.
